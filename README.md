# The Loop Clothing Exchange Data Visualization

Data visualization project for the GW LAI Data Visualization Competition 2026, analyzing customer visit data from The Loop clothing exchange program at George Washington University (Mount Vernon Campus).

![Dashboard](plot/loop-clothing-viz.png)

## Quick Start

```bash
Rscript code/loop-clothing-viz.R
```

**Requirements:** R with `readxl`, `tidyverse`, and `patchwork` packages.

## About The Loop

The Loop is a free clothing exchange program on GW's Mount Vernon Campus where students, staff, and faculty can take donated clothing items. This visualization explores visitor patterns, item preferences, and donation behavior from January–October 2025.

## Project Structure

```
├── code/loop-clothing-viz.R    # Visualization script
├── data/                       # Dataset (877 visits)
└── plot/                       # Generated visualizations
```
